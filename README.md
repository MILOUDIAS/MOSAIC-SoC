# MOSAIC-SoC

MOSAIC-SoC: Agent-Driven Multi-Core SoC Generator in Open-Source EDA.

## Project Plan

The project will be built in two steps:

1. **Phase 1 – Reconfigurability**
   - Enable SoC reconfigurability through a configuration file (YAML, Python, or similar).
   - Use this configuration to describe and generate SoC variants.

2. **Phase 2 – Agentic Approach**
   - Build a harness for SoC generation.
   - Equip the harness with skills related to the VLSI design flow, SoC design, and MOSAIC-SoC project-specific tasks.
